About axmol

axmol is a open source project is designed to be a cross-platform 2D game engine for building 2D games, demos and other graphical/interactive mobile applications. It runs on top of OpenGL 3.3(ES 2.0)/Metal and is written in C++.
This project is a fork of "cocos2d-x-4.0".
